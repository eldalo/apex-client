# APEX Client

## Description

API Client

- based on the [NestJS](https://nestjs.com/) framework.

## Environment Variables

| Variable Name      | Default Value  | Required |
| ------------------ | -------------- | -------- |
| `MONGODB_USERNAME` | `apex_user`    | Yes      |
| `MONGODB_PASSWORD` | `abc123qwerty` | Yes      |
| `MONGODB_DATABASE` | `db_apex`      | Yes      |

## Project setup

```bash
$ npm install
```

## Running MongoDB

```bash
$ docker compose up -d && docker compose start
```

## MongoDB CLI - create user and database

```bash
use db_apex
db.createUser({ user: "apex_user", pwd: "abc123qwerty", roles: [ { role: "readWrite", db: "db_apex" } ]})
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
